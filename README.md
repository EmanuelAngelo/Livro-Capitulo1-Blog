# Como executar o projeto

## Back end
Pré-requisitos: Java 11

```bash
# clonar repositório
git clone https://github.com/EmanuelAngelo/Blog-Cap1.git

# entrar na pasta do projeto 
# criar pasta venv
# python -m venv venv
# source venv/scripts/activate


# executar o projeto
python manage.py runserver
```
